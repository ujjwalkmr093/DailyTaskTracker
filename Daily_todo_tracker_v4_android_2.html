Export — backup your data
Export saves ALL your data as a JSON file to your computer. Save this file to Google Drive, a USB drive, or email it to yourself. If you switch browsers or computers, import this file to restore everything.
Import — restore from backup
Warning: Importing will merge with existing data. New data from file will be added. Existing data will not be deleted unless you clear first.
How your data is protected
Auto-save on every action
Every time you log anything, it saves instantly to browser localStorage
Browser localStorage is persistent
Data stays even if you close the browser or shut down your computer
What CAN lose your data
Clearing browser cache/history, switching browsers, reinstalling OS, or using incognito mode
Best practice — weekly export
Export backup every Sunday. Save to Google Drive folder named "Life Tracker Backups"
